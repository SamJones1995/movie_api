Horror Movies API

This API will provide a searchable database on Horror movies made since the year 2000. It will feature searches by actor and subgenre.

table showing URL endpoints and request methods
Request URL Response
Return a list of all movies(GET): Request body-none /horrorMovies JSON object showing a list of all movie titles
Return data on one movie(GET): Request body-none /horrorMovies/:title JSON object showing data on movie defined in URL
Return data about a subgenre(GET): Request body-none /subgenres/:name JSON object showing subgenre description
Return data about a director(GET): Request body-none /directors/:name JSON object showing director bio, birth year, death year
Allow new user to register(POST): Request body- JSON object with new user info EX { username: 'Test1', password: 'Test1', } /users JSON object showing new user info just added
Allow new user to update username(PUT): Request body-none /users/:username Text message showing showing the user information with new username
Allow new user to add a movie to their list of favorites(PUT): Request body-JSON object with movie title to be added to favorites EX 'The Ritual' /users/:username/favorites/:title Text message showing that the movie has been added to favorites
Allow new users to remove movie from favorites(DELETE): Request body-none /users/:username/favorites/:title Text message showing movie has been deleted from favorites
Allow users to deregister their account(DELETE): Request body-none /users/:username Text message showing username that has been deleted
