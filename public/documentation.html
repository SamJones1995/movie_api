Horror Movies API

This API will provide a searchable database on Horror movies made since the year 2000. It will feature searches by actor and subgenre.

table showing URL endpoints and request methods
Request URL Response
Return a list of all movies(GET): Request body-none /horrorMovies JSON object showing a list of all movie titles
Return data on one movie(GET): Request body-none /horrorMovies/:Title JSON object showing data on movie defined in URL EX: { "Genre": { "Name": "Animated", "Description": "example description" }, "Director": { "Name": "Victor Cook", "Bio": "example bio", "Birthyear": "1960", "Deathyear": "2021" }, "_id": "61a6ea6a134d6f40c5d220f0", "Description": "exmple description", "ImagePath": "hellboyanimated.png", "Featured": false, "Title": "Hellboy Animated: Blood and Iron" }
Return data about a subgenre(GET): Request body-none /horrorMovies/Genres/:Name JSON objects showing movie data under requested genre EX:{ "Genre": {
"Name": "Animated", "Description": "example description" }, "Director": { "Name": "Victor Cook", "Bio": "example bio", "Birthyear": "1960", "Deathyear": "2021" }, "_id": "61a6e926134d6f40c5d220ef", "Description": "example description", "ImagePath": "dantesinferno.png", "Featured": false, "Title": "Dante's Inferno"}
Return data about a director(GET): Request body-none /horrorMovies/Directors/:Name JSON object showing director bio, birth year, death year, and all directed movies
Allow new user to register(POST): Request body- JSON object with new user info EX {
"Username": "Test1",
"Password": "Test1",
"email": "email@email.com",
"Birthdate": "1992-01-01"
}		 /users JSON object showing new user info just added {
"Username": "Test1",
"Password": "Test1",
"email": "email@email.com",
"Birthdate": "1992-01-01"
"FavoriteMovies": [] }
Allow new user to update user info(PUT): Request body-json object with new user info EX: {
"Username": "Test1",
"Password": "Test1",
"email": "email1234@email.com",
"Birthdate": "1992-01-01"
}		 /users/:Username JSON object showing current user info after update {
"Username": "Test1",
"Password": "Test1",
"email": "email1234@email.com",
"Birthdate": "1992-01-01"
"FavoriteMovies": [] }
Allow new user to add a movie to their list of favorites(PUT): Request body-none /users/:Username/movies/:MovieID JSON object of current user info including newly added favorite movie
Allow new users to remove movie from favorites(DELETE): Request body-none /users/:Username/movies/:MovieID JSON object of current user info including newly deleted favorite movie
Allow users to deregister their account(DELETE): Request body-none /users/:Username Text message showing username that has been deleted
